# esg-simulator

Это игровой симулятор. 
Все персонажи, события и метрики являются вымышленными.
Не предназначено для оценки реальных компаний.

Данное программное обеспечение является вымышленной игрой-симулятором. Все совпадения с реальными компаниями, организациями, людьми или событиями являются случайными. Проект создан исключительно в развлекательных целях и не отражает реальных бизнес-процессов, экологических стандартов или управленческих практик каких-либо существующих юридических лиц.
